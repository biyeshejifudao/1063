# 基于协同过滤算法图书推荐系统

## 项目简介
图书推荐系统是基于协同过滤算法的推荐系统，功能上包含：1.用户登录注册  2.书籍管理  3.书籍推荐  4.订单管理 5.分类管理 6.用户评分包含前台系统和后台系统，前台系统对书籍进行展示、下单、推荐  后台系统：对书籍、用户、订单、地址进行管理。
